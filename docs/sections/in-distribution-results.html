Generalizing to Out-of-Distribution Games

To evaluate out-of-distribution generalization, we test ViGaL on AtariGPT, a benchmark for evaluating MLLMs as decision-making agents.

Compared to its base model (Qwen2.5-VL), ViGaL demonstrates significant cumulative reward improvement on Atari games despite being trained only on Snake and Rotation games. This is particularly notable because Atari games differ substantially from our training games in both visual appearance and gameplay strategies. These results suggest that ViGaL has strong out-of-distribution generalization to entirely unseen game environments.
